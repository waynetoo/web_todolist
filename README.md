# 待办事项应用

一个功能完整、界面美观的待办事项管理应用，使用纯HTML、CSS和JavaScript构建，集成了Supabase作为后端数据库。

## 功能特性

### 核心功能
- ✅ **添加待办事项** - 快速添加新的任务
- ✏️ **编辑待办事项** - 点击编辑按钮修改任务内容
- 🗑️ **删除待办事项** - 移除不需要的任务
- ☑️ **标记完成状态** - 勾选复选框标记任务完成状态

### 高级功能
- 🔍 **筛选功能** - 查看全部、待完成或已完成的任务
- 📊 **统计信息** - 实时显示总计、已完成和待完成数量
- 💾 **数据持久化** - 使用Supabase保存数据，支持多设备同步
- 🧹 **批量清理** - 一键清除所有已完成的任务
- ⌨️ **键盘快捷键** - 支持快捷操作提升效率
- 🔐 **用户认证** - 支持用户注册和登录
- 🔄 **实时同步** - 数据变更实时同步到所有设备
- 📱 **跨设备访问** - 在任何设备上访问您的待办事项

### 用户体验
- 🎨 **现代化设计** - 采用渐变背景和卡片式布局
- 📱 **响应式设计** - 完美适配桌面和移动设备
- ✨ **流畅动画** - 添加、删除任务时的平滑过渡效果
- 🎯 **直观交互** - 悬停效果和视觉反馈

## 技术栈

- **HTML5** - 语义化标签结构
- **CSS3** - 现代样式和动画
- **Vanilla JavaScript** - 纯JavaScript实现，无框架依赖
- **Supabase** - 后端即服务，提供数据库和认证
- **Google Fonts** - Inter字体

## 快速开始

### 1. 创建Supabase项目

1. 访问 [Supabase官网](https://supabase.com)
2. 点击 "Start your project" 或 "Sign Up"
3. 使用GitHub账号登录或注册新账号
4. 点击 "New Project"
5. 选择组织（或创建新组织）
6. 设置项目名称（例如：todo-app）
7. 设置数据库密码（请记住此密码）
8. 选择地区（选择离您最近的地区）
9. 点击 "Create new project"

### 2. 创建数据库表

1. 项目创建完成后，进入仪表盘
2. 点击左侧菜单的 "SQL Editor"
3. 点击 "New query"
4. 复制并粘贴 `create_todos_table.sql` 文件中的SQL代码
5. 点击 "Run" 执行SQL脚本

或者，您可以打开 `create_table.html` 页面，按照页面上的说明操作。

### 3. 获取API密钥

1. 点击左侧菜单的 "Settings"
2. 选择 "API"
3. 复制以下信息：
   - Project URL（项目URL）
   - anon public key（匿名公开密钥）

### 4. 配置应用

1. 打开 `supabase-config.js` 文件
2. 将以下变量替换为您的项目信息：
   ```javascript
   const SUPABASE_CONFIG = {
       URL: 'YOUR_SUPABASE_URL',
       ANON_KEY: 'YOUR_SUPABASE_ANON_KEY'
   };
   ```

### 5. 运行应用

1. 在项目根目录运行一个简单的HTTP服务器：
   ```
   python -m http.server 8000
   ```
2. 在浏览器中访问 `http://localhost:8000`

## 使用方法

1. **打开应用** - 在浏览器中打开应用URL
2. **注册/登录** - 使用邮箱注册新账户或登录现有账户
3. **添加任务** - 在输入框中输入任务内容，设置可选的日期、时间和优先级，然后点击"添加"按钮
4. **管理任务** - 使用复选框标记完成，点击删除按钮移除任务
5. **筛选查看** - 使用顶部的筛选按钮查看不同状态的任务
6. **排序任务** - 使用排序选项按不同方式排序任务
7. **批量清理** - 点击"清除已完成"按钮删除所有已完成的任务

## 键盘快捷键

## 故障排除

### 问题：获取待办事项失败: {code: PGRST205, details: null, hint: null, message: Could not find the table 'public.todos' in the schema cache}

**解决方案：** 这表示Supabase数据库中还没有创建todos表。请按照"创建数据库表"部分的说明创建表。

### 问题：CORS错误

**解决方案：** 在Supabase设置中添加您的域名到CORS允许列表。

### 问题：数据不同步

**解决方案：** 检查实时订阅是否正确设置，确保您的Supabase项目已启用实时功能。
